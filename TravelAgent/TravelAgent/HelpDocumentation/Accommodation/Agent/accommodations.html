Accommodation Help

Understanding "Accommodation" Page

On this page, you can find all the information related to accommodation. Each accommodation is accompanied by a picture, name, average rating, and location.

By clicking on the blue icon in the bottom right corner, you can search accommodations by specific fields, which are name,address and rating. You can specify lower bound and upper bound for rating.


Add

Clicking on the highlighted button will open the accommodation creation window



It is necessary to enter a picture of the accommodation, the name of the accommodation, rating and the location where the accommodation is located (select from the map).


Edit

After selecting an accomodation, clicking on the highlighted button will open the accommodation editing window


This option enables you to change already existing values set for an accomodation, all of the fields are the same as in the creation window just with already set values. To see field explanations go ti Add


Remove

After selecting an accomodation, clicking on the highlighted button will open the deletion dialog, asking you if you wanted to permanently delete this accomodation
